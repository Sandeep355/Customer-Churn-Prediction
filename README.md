# 📊 Customer Churn Prediction & Retention Intelligence Platform

## 📌 Overview

This project is an end-to-end Machine Learning application that predicts whether a customer is likely to churn. It includes data preprocessing, feature engineering, model training, explainable AI, and a Streamlit web application for interactive predictions.


## 🚀 Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Multiple Machine Learning Models
- Model Evaluation
- Explainable AI (SHAP)
- Streamlit Web Application
- Download Prediction Results
- Interactive Visualizations


## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- SHAP
- Streamlit
- Plotly
- Joblib
- Jupyter Notebook
- Power BI

## ▶️ How to Run the Project

### Clone the repository

```bash
git clone <repository-link>
```

### Navigate to the project folder

```bash
cd Customer-Churn-Prediction
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the Streamlit application

```bash
streamlit run app/app.py
```


## 📈 Results

The application predicts customer churn probability and categorizes customers into:

- 🟢 Low Risk
- 🟡 Medium Risk
- 🔴 High Risk

It also provides downloadable prediction results and interactive charts.
